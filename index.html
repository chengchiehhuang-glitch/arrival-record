一般員工不需要填這個。只有管理員在重新部署 Apps Script 後才需更新。

📋 Apps Script 部署步驟
  1. 開啟 Google Sheets 新建試算表，命名「到貨記錄」
  2. 點上方「擴充功能」→「Apps Script」
  3. 刪除預設程式碼，貼上下方程式碼
  4. 按「部署」→「新增部署作業」→ 類型選「網頁應用程式」
  5. 執行身分選「我」，存取權選「所有人」→「部署」
  6. 複製 Web App URL 貼入上方欄位
